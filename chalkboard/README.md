# Chalkboard Pack — KOOSY

## 촬영 루틴 10줄

1. **칠판 열기** — `index.html` 실행 또는 https://dtslib1979.github.io/koosy/chalkboard/
2. **오늘 주제 확인** — Topic 섹션 읽기
3. **카드 훑기** — 6개 카드 클릭해서 자료 확인
4. **REC 누르기** — 칠판 상단으로 스크롤
5. **촬영 시작** — 화면 녹화 또는 카메라 ON
6. **칠판 보며 말하기** — 카드 순서대로 설명
7. **촬영 종료** — 녹화 정지
8. **Quick Access → Archive** — 결과물 저장
9. **Quick Access → Engine** — HQ 발행 도구 이동
10. **내일 board.json 업데이트** — HQ Brief 반영
